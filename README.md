# GalleryApp

## Overview
**GalleryApp** is a cross-platform photo gallery built with .NET MAUI. It fetches images from Unsplash, supports favorites, and provides a detail view with author info.

---

## Features
- **Feed:** Infinite scrolling photo list with lazy loading
- **Favorites:** Add/remove with persistent storage (Preferences)
- **Details:** Author name, external link, and back navigation
- **Responsive UI:** Adaptive layouts for mobile screens

---

## Tech stack
- **Framework:** .NET MAUI
- **Architecture:** MVVM (CommunityToolkit.Mvvm)
- **UI:** XAML
- **Routing:** Shell navigation
- **API:** Unsplash

---


## Getting started
1. **Prerequisites:** Visual Studio 2022 with .NET MAUI workload
2. **Clone:**
   ```bash
   git clone https://github.com/username/GalleryApp.git
   cd GalleryApp
3. Run:

    Windows/macOS: select GalleryApp startup project, press Run

    Android/iOS: select an emulator or a real device and press Run
